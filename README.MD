# 微博互关检测脚本

![demo](pic/demo.png)

本脚本作者：Poc-Sir，兼容Python3.X环境，需安装requests库：
`pip3 install requests`

本脚本需用户在“weibo.cn”（彩版）中登录，注意不是“weibo.com”：

![login](pic/login.png)

并向脚本提供登陆之后Cookie中的SUB数据内容：

![cookie](pic/cookie.png)
